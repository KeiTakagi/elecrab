えれくらぶ サンプルプログラム for Arduino
======================
「えれくらぶ」は、エルチカの次の一歩が踏み出せない悩みを解決するために製作しました。  
サーボモータ２個とマイコンボードで、前後、左前後、右前後の６方位に動かすことができます。  

プログラムを組んだり、赤外線リモコンを使って楽しんでください。
